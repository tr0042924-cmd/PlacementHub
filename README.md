# рџ“– Overview

PlacementHub is a **Full Stack Placement Management Portal** developed to simplify and automate the campus recruitment workflow.

The platform provides dedicated portals for **Students**, **Recruiters**, and **Administrators**, enabling seamless job posting, application tracking, resume management, and placement monitoring through a centralized system.

The project focuses on improving transparency, reducing manual effort, and providing a smooth recruitment experience for educational institutions.
